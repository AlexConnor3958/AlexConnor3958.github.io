FOGGS - Pacman:

In our FOGGS sessions we were asked to create a single project in C++ over the course of the first semester.
(Although I am planning on continuing with this project in my spare time to polish it up and add a few things)

In our FOGGS module for the first semester we were asked to use a given framework and some tutorials to create a 2D game. Originally all of the tutorials created a basic Pacman style game hence the name. However we were asked to take what we had learned and work some things out for ourselves and change the style of the game from Pacman. I went with a game kind of like the 'Journey of the Prairie King' minigame from Stardew Valley to hand in. I hope the version that I continue working on comes out more like Hotline Miami.

This game is played using the WASD keys and the mouse with the player looking at the mouse and shooting the semi - automatic pistol on a left click. The player can move around not confined by the walls instead they wrap around and appear on the other side. The player has to fight through 5 different waves of different numbers of enemies all coming from different angles. The players pistol has a magazine of 18 rounds and they start with no extra. When an enemy dies they drop an ammo pack which gives the player a random amount of ammo between 1 and 10. The player reloads by pressing 'r' and tries to get through the most rounds gaining points as they go and try to get a highscore which will be saved to a text file stored in the project folder.

Rapid - Top Down Shooter:

For the last five weeks of the year for Rapid we were asked to make a top down shooter.
This is the last thing I made in Unity for this year and so consolidated all of my knowledge in Unity and I think it turned out to be my best Rapid project.

For the final project in Rapid for this year we were given the least fleshed out framework to work on and asked to create a top down shooter. The tutorial we were given also had almost no code and so we needed to use all that we learned over the year and make a game. I wanted to create a game alot like 'Bounty Of One' which is a fast paced roguelite bullet hell. If I were to improve this I would include some ranged enemies to add a bit of variation and maybe add some player feedback for what upgrade is selected and in the game overall. I would also balance the automatic rifle as it is currently really underpowered.

This game is played using the WASD keys to move, the arrow keys to shoot using the mouse to choose the upgrades. The player can move around in the arena defined by the walls and shoot at the enemies using the three different classes given at the start of the game. There is no loading or ammo requirements for the weapons but there is player health. As the player kills enemies they will drop exp and health orbs. The health orbs heal the player and the exp orbs add some exp to the players level. Once the player levels up, they get a choice of three upgrades which will upgrade the weapon or character.
