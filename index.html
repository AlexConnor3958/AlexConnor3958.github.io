FOGGS - Pacman:

In our FOGGS sessions we were asked to create a single project in C++ over the course of the first semester.
(Although I am planning on continuing with this project in my spare time to polish it up and add a few things)

In our FOGGS module for the first semester we were asked to use a given framework and some tutorials to create a 2D game. Originally all of the tutorials created a basic Pacman style game hence the name. However we were asked to take what we had learned and work some things out for ourselves and change the style of the game from Pacman. I went with a game kind of like the 'Journey of the Prairie King' minigame from Stardew Valley to hand in. I hope the version that I continue working on comes out more like Hotline Miami.

EMBEDDED VIDEO OF GAME

This game is played using the WASD keys and the mouse with the player looking at the mouse and shooting the semi - automatic pistol on a left click. The player can move around not confined by the walls instead they wrap around and appear on the other side. The player has to fight through 5 different waves of different numbers of enemies all coming from different angles. The players pistol has a magazine of 18 rounds and they start with no extra. When an enemy dies they drop an ammo pack which gives the player a random amount of ammo between 1 and 10. The player reloads by pressing 'r' and tries to get through the most rounds gaining points as they go and try to get a highscore which will be saved to a text file stored in the project folder.
